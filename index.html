Portafolio

Programa de Administración de Rentas con Python y Flask

Sistema web integral para la administración de propiedades de alquiler, desarrollado con Python y Flask. Esta aplicación robusta centraliza la gestión de propiedades, inquilinos y contratos, automatizando el seguimiento de pagos y la generación de reportes. Incluye un panel de control intuitivo que ofrece una visión clara del estado de las rentas y facilita la toma de decisiones. La arquitectura se apoya en PostgreSQL para garantizar la integridad y seguridad de los datos.

Sistema de Gestión de Inventario (Python GUI con BD)

Aplicación de escritorio diseñada para la gestión eficiente de inventarios. Desarrollada en Python con una interfaz gráfica (GUI) intuitiva, permite un control detallado de productos, categorías y stock. El sistema se conecta a una base de datos relacional para asegurar la persistencia de los datos, registrando cada movimiento de entrada y salida. Es una solución ideal para pequeños y medianos negocios que buscan optimizar su logística y control de existencias de manera efectiva.

Página de la NASA con Flask

Aplicación web interactiva que consume la API de la NASA para explorar el cosmos. Permite a los usuarios descubrir la 'Foto Astronómica del Día' (APOD), obtener información sobre objetos cercanos a la Tierra y explorar una vasta galería de imágenes del espacio. Desarrollada con Python y Flask, esta aplicación ofrece una experiencia de usuario fluida y educativa, acercando las maravillas del universo a través de una interfaz atractiva y fácil de usar.

Juego en C#

Juego de rol y exploración de mazmorras (dungeon crawler) desarrollado en C#. Este proyecto aplica la Programación Orientada a Objetos (POO) para modelar el comportamiento de personajes, la IA de los enemigos y la generación procedural de niveles. La arquitectura del juego está diseñada para ser extensible, permitiendo la fácil adición de nuevos contenidos como ítems, habilidades y monstruos, ofreciendo una base sólida para un juego completo.
